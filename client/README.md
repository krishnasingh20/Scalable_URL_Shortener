# Scalable URL Shortener

A scalable URL shortening platform built using React.js, Node.js, Express.js, MongoDB, and Redis. The application allows users to generate short URLs, redirect users efficiently, and track basic analytics with optimized backend performance.

## Features

- Shorten long URLs into compact shareable links
- Fast URL redirection
- Redis caching for improved performance
- RESTful API architecture
- Basic click analytics
- Rate limiting for API protection
- Input validation and error handling
- Modular backend structure
- Responsive React frontend

## Tech Stack

- Frontend: React.js, HTML, CSS
- Backend: Node.js, Express.js
- Database: MongoDB
- Caching: Redis
